# Tecnología de Control Automotriz

Curso técnico orientado a la formación de estudiantes del ciclo superior en la carrera de Técnico Automotor. El programa desarrolla una base sólida en electricidad, electrónica y sistemas de control, con una progresión hacia la comprensión e integración de sistemas automotrices reales.

## Propósito formativo

El curso busca que el estudiante no solo adquiera conocimientos teóricos, sino que desarrolle la capacidad de interpretar el comportamiento de sistemas técnicos, realizar mediciones, analizar señales y comprender la lógica de funcionamiento de los sistemas electrónicos aplicados al automóvil.

## Enfoque de enseñanza

La propuesta se basa en una secuencia progresiva que articula teoría, práctica y aplicación. Se parte de los fundamentos eléctricos, se avanza hacia los componentes electrónicos y se integra el uso de microcontroladores en la construcción de sistemas automatizados.

El aprendizaje se organiza a partir de:

- resolución de situaciones problemáticas  
- prácticas guiadas con distintos niveles de dificultad  
- simulación de circuitos  
- implementación en hardware real  
- vinculación constante con sistemas automotrices  

## Estructura del curso

Bloque 1 – Fundamentos eléctricos  
Magnitudes eléctricas, ley de Ohm, medición con multímetro y análisis de circuitos serie, paralelo y mixto.

Bloque 2 – Electrónica aplicada  
Estudio de componentes electrónicos y su comportamiento en circuitos. Aplicación en sistemas automotrices básicos.

Bloque 3 – Control electrónico  
Introducción al uso de microcontroladores, entradas y salidas, lógica de control y trabajo con sensores.

Bloque 4 – Sistemas automotrices integrados  
Desarrollo de sistemas que integran sensores, controladores y actuadores, replicando el funcionamiento de sistemas reales.

## Proyecto integrador

Como cierre del proceso formativo, se desarrolla un sistema de control de refrigeración automotriz. En este proyecto el estudiante integra:

- sensor de temperatura  
- controlador  
- relé  
- actuador (electroventilador)  

Este modelo permite comprender la relación entre medición, decisión y actuación dentro de un sistema técnico.

## Organización del contenido

Cada bloque incluye:

- presentaciones teóricas  
- infografías de apoyo  
- prácticas guiadas  
- actividades de aplicación  

El material está organizado de manera progresiva para facilitar la comprensión y la transferencia al entorno real.

## Aplicación

El curso reproduce situaciones habituales del diagnóstico técnico automotriz, permitiendo que el estudiante desarrolle criterios de análisis y toma de decisiones aplicables tanto en el ámbito educativo como en contextos de trabajo.
